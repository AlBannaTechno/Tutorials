Whilst this is "work in progress", you can use this page to practise writing and editing help pages. Adding text is a two step process:

1. Type the html format text with your favourite editor

2. Compile the text with pyuicc4 or better with the makepyqt.pyw. The latter can be downloaded with the sample chapters from http://www.qtrac.eu/pyqtbook.html
